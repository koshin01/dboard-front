# 👀 **DBoard Frontend**

## **About**
オープンソースプロジェクト「DBoard」のフロントのリポジトリです。<br>
コントラクトのリポジトリと合わせて、DBoard は動いています！

**プログラミング言語**<br>

**ライブラリ**<br>

- ethers.js<br>

**フレームワーク**<br>

- Headless UI<br>
